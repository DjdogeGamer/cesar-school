A turma será dividida em 2 grupos. Cada grupo deverá propor (implementar) um método de
criptografia simétrica que utilize substituição e/ou permutação. Uma chave de 16 bits
(equivalente a dois caracteres) deverá ser utilizada para encriptar a mensagem. A chave, o
texto claro e o texto criptografado deverão ser enviados para o professor sem o conhecimento
do outro grupo. Cada grupo deverá tentar quebrar a cifra do outro grupo.

Etapas:
1. Apenas o texto cifrado é conhecido pelo atacante (além do tamanho da chave);
2. Os algoritmos passam a ser conhecidos - o ataque deve focar em criptoanálise e não
em força bruta;
3. Mensagens estruturadas passam a ser utilizadas (texto claro escolhido).
